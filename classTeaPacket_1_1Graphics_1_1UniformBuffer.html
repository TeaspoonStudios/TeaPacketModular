◆ ShouldUBBeEndianSwapped

const bool TeaPacket::Graphics::UniformBuffer::ShouldUBBeEndianSwapped
static

Whether the data inside a UniformBuffer needs to be Endian Swapped when sending the data over.

Certain platforms (Wii U) require the data within a Uniform Buffer to be endian swapped before being sent over. The user must do this swapping themselves, as UniformBuffers know nothing about the data within them.
