Detailed Description

Represents a user-viewable area.

On PC-based implementations, this typically represents a window. On non-PC implementations, this typically represents a screen.

static void TeaPacket::Graphics::Display::InitializeDefaultDisplays ( const std::vector< DisplayParameters > & requestedParameters)
static

Creates any number of displays, based on the platform.

This function will always create at least 1 display. On PC platforms, this function will create a single window to be rendered to. On other platforms, this will create a Display for every screen that can be rendered to.
