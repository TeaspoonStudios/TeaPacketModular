Detailed Description

A module for rendering and presenting graphics to the user.

This module specifically represents a shader-based, OpenGL style rendering engine. Modules may support newer Graphics APIs, but any hardware lacking shader support will not be supported.
This module may be renamed at a future date to a more descriptive name once fixed-function and Vulkan-style modules exist.

◆ TextureWrapMode

enum class TeaPacket::Graphics::TextureWrapMode : unsigned char
strong

Determines how a texture is used when UV coordinates fall outside the 0-1 range.

Enumerator
Wrap 

This texture wraps around starting from 0 after passing 1. Also known as Repeat.

Clamp 

This texture uses the closest value from the 0-1 range after passing 0 or 1, causing stretching when past the 0-1 range.

Mirror 

This texture is mirrored and repeats every time a 0-1 border is passed.

◆ DrawMesh()

void TeaPacket::Graphics::DrawMesh ( )

Draws a mesh to the screen, using any and all active graphics data.

This function heavily depends on several "active" variables/states. These are:

  • The active Mesh
  • The active Shader
  • The active Texture (s)
  • The active UniformBuffer (s).
  • The active Viewport.
